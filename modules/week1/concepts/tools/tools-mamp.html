# MAMP

MAMP is a one-click-solution for setting up your personal webserver. MAMP installs a local server environment in a matter of seconds on your computer. It comes free of charge, and is easily installed. It is available for both Windows and macOS.

Installing MAMP

To install MAMP go to www.mamp.info to download and install the latest version of MAMP.  During the installation processes, you will have an option to install MAMP Pro. This is the default. I recommend you uncheck this option. 

Here are the full instructions for installing MAMP on a Mac and installing MAMP on a Windows machine.

Configuring MAMP

The free version of MAMP has limited options for customization, but there are a few things that needs to be configured. 

Start/Stop Servers

It is best to have MAMP to start the Apache and MySQL Servers when MAMP is run. This is set in the MAMP preferences under Start/Stop Tab. It is also good to stop the servers when MAMP is quit. 

Configure Ports

By default, the MAMP Apache server will run on port 8888 and the MySQL server on port 8889. These ports are defaulted to limit the chance of interference with other tasks and applications that might be running on your machine. However, this not the standard ports for Web and MySQL servers, 80 and 3306 respectively. 

To use the standard ports, go to the MAMP preferences, under the Ports tab, and click the "Set Web & MySQL ports to 80 & 3306" button. 

PHP Version

The free version of MAMP typically comes with two versions of PHP, which at the time of this writing is 7.0.20 and 7.1.6. We will be working with edumedia this semester, which is running PHP version 7.0.12. So to limit any issues with the different versions PHP, I recommend that you set MAMP to version 7.0.x. 

This can be done from the MAMP preferences, under the PHP tab.

Setting the Document Root

The Document Root is the directory that Web Server will look for files. This is also the directory that you will store all files associated with any PHP project.  By default MAMP sets the Document Root in /Applications/MAMP/htdocs/ on macOS and C:\MAMP\htdocs on Windows. However, it is possible to set the Document Root to anywhere you wish. 

To change the location of the Document Root, to the MAMP preferences and select the Web Server tab. For macOS, click the folder icon and select the directory you wish to be the Document Root. For Windows, click the "Select..." button and select the directory you wish to be the Document Root.
