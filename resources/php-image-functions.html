# Working with Images

PHP is not limited to just HTML output, it is also possible to use PHP to create a manipulate image file in a variety of different formats, including GIF, PNG, and JPEG. PHP can even output image streams directly to a browser. This is possible using the PHP GD Extension Library and Image Functions.

In this course we will be only covering and using a small number of functions, so please refer to the Image Processing and GD for more information about the different functions and capabilities of the GD Extension Library.

# Checking for GD Installation

# NEED TO BE UPDATED

The GD Extension Library may not always been installed on a web server. So, it is necessary to check when working on a new environment. The easiest way to check is to use the phpinfo() function. When viewing the page, search for gd to verify that the extension is installed and enabled. By default the GD Extension Library is installed and enabled on MAMP.

# Image Functions

# getimagesize

The getimagesize() function return an array of information about the size of image. The function required a single parameters, a string containing the path to the file you wish to retrieve information about.

{% highlight php linenos %} <?php $size = getimagesize('images/image.jpg'); print_r($size); ?> {% endhighlight %}

# imagecreatefromjpeg

The imagecreatefromjpeg() function create a new image from an existing jpg file and stores it in memory, or in other words, it make a copy. The copy can be manipulated, saved, or destroyed without affecting the original. The function required a single parameters, a string containing the path to the jpg file you wish to copy. If successful, the function returns a image resource id, otherwise the function will return false.

{% highlight php linenos %} <?php $copy = imagecreatefromjpeg('images/image.jpg'); echo $copy; ?> {% endhighlight %}

# imagejpeg

The imagejpeg() function output a jpeg directly to the browser OR save the image to a file. The function can take three parameters, an image resource, the path to where the image should be save (if null, a image will be output to the browser directly), and the quality of the image from 0 (worst) to a 100 (best).

{% highlight php linenos %} <?php $copy = imagecreatefromjpeg('images/image.jpg'); imagejpeg($copy); ?> {% endhighlight %}

# imagedestroy

The imagedestroy() function deletes a image resource from memory. It requires one parameter, an image resource, and returns true if it successful destroyed the image and false if it is not.

{% highlight php linenos %} <?php $copy = imagecreatefromjpeg('images/image.jpg'); imagejpeg($copy); imagedestroy($copy); ?> {% endhighlight %}

# imagescale

The imagescale() function will scale an image using the provided width and height. The function take four parameters, an image resource, the new width, the new height (if omitted or negative, the original aspect ratio will be preserved), and the mode or the algorithm used to scale the image. If successful, the function will return an image resource, otherwise it will return false.

{% highlight php linenos %} <?php $copy = imagecreatefromjpeg('images/image.jpg'); $copy = imagescale($copy); imagejpeg($copy); imagedestroy($copy); ?> {% endhighlight %}
