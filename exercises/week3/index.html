# Clue

# Understanding the Problem

Before we begin, we must understand the problem. There are 9 rooms in the Clue Mansion. Each mansion may contain a weapon and / or suspect. But the sake of our project, we can think of each room, as a different page in our site, the items in room is merely the contents on the page.

# Updating the Room Links

Create the rooms array was necessary to each of the pages content in a format that PHP understands. But having the content is only the first step. Now we need to find a way to create a page for each room.

Remember back to our example of this project using HTML. We would view a different room by going a different URL. For example, if we want to see the ballroom we went to ballroom.html. That is all we are really doing here except, instead of changing the URL by going to a new page, we are going to do it change the query string.

The query string is a set of key / value pairs that follow a question mark (?) after the page in the URL, and look something like this:

index.php?room=Ballroom

So, from our HTML example, we will need to convert all the room links from pointing to HTML pages to use a query string instead.

For example, this link:

<a href="ballroom.html">Ballroom</a>

would become this:

<a href="?room=Ballroom">Ballroom</a>

Now, we could update all the link manually inside the HTML, but, since we already have an array of all of the rooms it would be more efficient to use the $rooms array and a foreach to recreate the links dynamically.

<main class="container">
  <?php foreach ($rooms as $room => $items) : ?>
    <div class="room">
      <a href="?room=<?php echo $room; ?>">
        <?php echo $room; ?>
      </a>
    </div>
  <?php endforeach; ?>
</main>

# Using the Query Strings

A query string is just a list of key / value pairs. These key / value pairs are sort of like sending variables through the URL. When these variables or query strings are set, PHP will automatically collect each key / value pair and store them in the $_GET superglobal array.

Superglobals

Superglobals are built-in variables that are always available in all scopes

From our query string example above, we can expect PHP to create an item in the $_GET array with an index of room and a value of ballroom. We would access the room index from the $_GET just like we would from any other array.

$_GET['room']; // ballroom

Of course, because the query string is part of the URL it is vulnerable to tempering by the player. So, limit the potential for errors we will to add a couple of checks before we use the value of $_GET['room']. We will first check if the is, in fact, an index of room in the $_GET and then we will check if the value of $_GET['room'] is one of the rooms in our $rooms array. In both cases, the isset() function can be used to see if an index in array exists or is set.

isset

The isset() function determines if a variable is set and is not NULL.

These checks can be done all in a single if statement with use of the AND (&&) logical operator.

if (isset($_GET['room']) && isset($rooms[$_GET['room']])) {

}

Now that we have confirmed that we have a valid room, we will want get the items for that room. We can do this by using the $rooms array and $_GET['room'] as the index to get the desire room. From there we would retrieve the items array.

NOTE

Remember that when dealing with multidimensional arrays, we need an additional set square brackets for each nested array.

if (isset($_GET['room']) && isset($rooms[$_GET['room']])) {
    $items = $rooms[$_GET['room']]['items'];
}

# Displaying the Items

The final part of the project is to display a message about what, if anything was found in the room. There are three possible messages:

// If the room has one or more items
You found item and item.

// If the room has no items
You found no items

// If no room or an invalid room is selected
Select a room to get a clue.

To accurately display all three messages, we will use a set of nested if / else statements.

First we will check if a valid room has been given, buy check $_GET['room'], but we already did that to get the items array and set $items. So, we can use a shortcut and just check if $items has been set, and if it is not set display the "Select a room..." message.

<div class="items"> 
  <?php if (isset($items)) : ?>
      
  <?php else : ?>
    <p>Select a room to get a clue.</p>
  <?php endif; ?>
</div>

Now, if $items is set, then we need check it the array actually has any items in it. We can do this by using the count() function. If the count is 0 we know it has no items in it.

count

The count() function counts all elements in an array, or something in an object.

<div class="items"> 
  <?php if (isset($items)) : ?>
      <?php if (count($items)) : ?>

      <?php else : ?>
        <p>You found no items.</p>
      <?php endif; ?>
  <?php else : ?>
    <p>Select a room to get a clue.</p>
  <?php endif; ?>
</div>

Finally, if the $items array does contain items, we will want to display the items as a string the following syntax.

You found Mr. Green and the Knife.

Now, it is possible to do this using a foreach loop, but in this case it is easier and more efficient to use the implode() function. The function takes two parameters a string to serve as the "glue" and the array.

implode

The implode() function joins array elements with a string.

<?php if (isset($items)) : ?>
  <?php if (count($items)) : ?>
    <p>You found <?php echo implode(' and ', $items); ?>.</p>
  <?php else : ?>
    <p>You found no items.</p>
  <?php endif; ?>
<?php else : ?>
  <p>Select a room to get a clue.</p>
<?php endif; ?>
